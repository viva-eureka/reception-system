## 受付端末シナリオ（来訪者側）

### US-01 招待済み来訪者がQRで受付する

**アクター**: A1  
**事前条件**: 招待メールが届いており、QRコードを画面または印刷で手元に持っている  

1. 来訪者が受付端末（iPad）の前に立つ
2. 待ち受け画面をタップ → 受付メニューへ
3. 「ご予約・招待の方」を選択
4. 「QRコードをかざす」を選択
5. カメラにQRコードをかざす
6. 本人確認 → 受付完了画面に氏名と「担当者がまいります」が表示される
7. Google Chat に担当者へ通知が飛ぶ
8. 入館記録が自動作成される

**成功条件**: 担当者が通知を受け取り、来訪者を迎えに来る  
**例外**: QRが読み取れない → キーボード入力に誘導する

---

### US-02 来訪者が名刺を撮影して受付する

**アクター**: A1 / A2  
**事前条件**: 名刺を持参している  

1. 受付メニューで「ご予約・招待の方」→「名刺を撮影」を選択
2. カメラガイド枠に名刺を合わせる
3. 「撮影する」をタップ → OCRで氏名・会社名を自動読み取り
4. 読み取り内容を画面で確認・修正
5. 「受付する」をタップ
6. 受付完了 → 担当者へ通知

**成功条件**: 氏名・会社名が正確に読み取られ、スムーズに受付完了  
**例外**: 読み取り精度が低い → 修正フォームで手入力して続行

---

### US-03 来訪者がキーボードで氏名を入力して受付する

**アクター**: A1 / A2  
**事前条件**: なし  

1. 受付メニューで「ご予約・招待の方」→「お名前を入力」を選択
2. ソフトキーボードで氏名（必須）・会社名（任意）を入力
3. 「受付する」をタップ
4. 受付完了 → 担当者へ通知

---

### US-04 配送業者がワンボタンで受付する

**アクター**: A3  
**事前条件**: なし  

1. 待ち受け画面をタップ
2. 「配送業者の方」ボタンをタップ
3. 受付完了画面が表示される（「担当者がまいります」）
4. Google Chat に「配送業者が到着」通知が飛ぶ
5. 配送記録が自動作成される

**成功条件**: 5秒以内に完了。来訪者入力が不要  

---

### US-05 営業時間外に来訪者が訪問する

**アクター**: A1 / A2 / A3  

1. 受付時間外（設定した開始〜終了時刻の外）に端末を操作
2. 待ち受け画面に「ただいま営業時間外です」と受付時間が表示される
3. 操作不可（受付フローに進めない）

## 管理・運用シナリオ（社内担当者・管理者側）

### US-06 担当者が来訪通知を受け取って来客を迎える

**アクター**: A4  
**事前条件**: Google Chat の該当スペースに参加している  

1. US-01〜04 のいずれかで来訪者が受付を完了する
2. Google Chat に通知が届く（氏名・会社名・受付時刻・会議室）
3. 担当者がエントランスへ迎えに行く

---

### US-07 受付管理者が来訪予約を作成し招待メールを送る

**アクター**: A5  
**事前条件**: 来訪者のメールアドレス・来訪日時が確定している  

1. 管理画面 → 招待管理 → 「新規招待を作成」
2. 来訪者名・会社名・メールアドレス・担当者・日時・会議室を入力
3. 「招待を作成」をクリック
4. QRコードが自動生成される
5. Googleカレンダーに予定・会議室予約が自動作成される（連携後）
6. 「招待メールを送信」をクリック → QRコード添付メールが送信される

**成功条件**: 来訪者が招待メールとQRコードを受け取る  

---

### US-08 受付管理者が入退館記録を確認する

**アクター**: A5  

1. 管理画面 → 入退館記録 を開く
2. 当日の入館・退館の一覧を時系列で確認
3. 在館中の来訪者を把握する
4. 必要に応じて退館ボタンを操作し退館記録を追加する

---

### US-09 受付管理者が受付設定を変更する

**アクター**: A5  

1. 管理画面 → 設定 を開く
2. 以下を必要に応じて変更する
   - 待ち受けテーマ（5種類）
   - 受付時間（開始・終了）
   - 受付完了画面の表示モード（プライバシー設定）
   - 会議室の追加・削除
   - Google Chat Webhook URL

---

### US-10 受付時間外設定を変更して祝日・休業に対応する

**アクター**: A5  

1. 管理画面 → 設定 → 受付時間を変更
2. 祝日前日は終了時刻を早める、など
3. 設定保存 → 即時反映
